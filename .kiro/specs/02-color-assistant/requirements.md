# Requisitos: Asistente de combinación de colores

## R1. Contexto

- CUANDO el usuario solicite una paleta, EL SISTEMA DEBERÁ considerar ambiente, tamaño, luz natural, luz artificial, piso, mobiliario, estilo y sensación.
- CUANDO el usuario descarte un color o familia, EL SISTEMA NO DEBERÁ incluirlo.
- SI faltan datos, EL SISTEMA DEBERÁ mostrar qué supuestos utilizó.

## R2. Recomendación

- EL SISTEMA DEBERÁ devolver color principal, secundario, acento y techo.
- EL SISTEMA DEBERÁ indicar proporciones y superficies sugeridas.
- EL SISTEMA DEBERÁ explicar por qué la paleta funciona en ese ambiente.
- EL SISTEMA DEBERÁ ofrecer al menos una variación más cálida, más clara o con diferente contraste.

## R3. Restricciones comerciales

- CUANDO una fórmula no pueda reproducirse con la base disponible, EL SISTEMA NO DEBERÁ presentarla como comprable.
- CUANDO falte stock, EL SISTEMA DEBERÁ diferenciar recomendación conceptual de propuesta disponible.
- EL SISTEMA DEBERÁ asociar colores únicamente con productos aptos para el uso y superficie.

## R4. Accesibilidad

- EL SISTEMA DEBERÁ evaluar contraste cuando la combinación afecte texto, señalización o navegación.
- EL SISTEMA NO DEBERÁ usar color como único indicador de estado.
