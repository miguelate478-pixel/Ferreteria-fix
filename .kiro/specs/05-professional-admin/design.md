# Diseño: Modo profesional y administración

## Experiencia profesional

Vista tabular de alta densidad con atajos, edición por lotes, copiar/pegar, plantillas e importación. El ambiente sigue siendo la unidad de trabajo; la tabla acelera, no reemplaza el modelo.

## Roles iniciales

- Cliente.
- Profesional.
- Vendedor.
- Administrador de catálogo.
- Administrador de inventario.
- Administrador de organización.
- Auditor/solo lectura.

## Validación de publicación

Los productos no se publican si faltan rendimiento, uso, superficies, manos, presentaciones o estado. Los colores no se ofrecen como comprables si no existe fórmula válida para una variante disponible.

## Analítica

Eventos de dominio anonimizables: `project_started`, `measurement_completed`, `palette_selected`, `quote_generated`, `quote_accepted`, `substitution_accepted`, `order_completed`, `reorder_started`.
