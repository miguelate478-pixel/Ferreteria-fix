# Requisitos: Modo profesional y administración

## R1. Profesional

- CUANDO un usuario profesional cree una obra, EL SISTEMA DEBERÁ permitir múltiples ambientes y superficies.
- EL SISTEMA DEBERÁ permitir copiar ambientes tipo, importar datos y editar en tabla.
- EL PROFESIONAL DEBERÁ poder aplicar desperdicios, productos y precios por convenio.
- EL SISTEMA DEBERÁ generar cotizaciones con o sin desglose interno según permisos.

## R2. Colaboración

- CUANDO se comparta un proyecto, EL SISTEMA DEBERÁ definir permisos de lectura, comentario, edición o aprobación.
- CUANDO un cliente apruebe una propuesta, EL SISTEMA DEBERÁ registrar versión, actor y fecha.

## R3. Administración

- EL ADMINISTRADOR DEBERÁ gestionar productos, fichas, colores, fórmulas, compatibilidades, reglas, precios, inventario y proveedores.
- CADA cambio crítico DEBERÁ quedar auditado.
- EL SISTEMA DEBERÁ impedir publicar fichas incompletas o inconsistentes.

## R4. Analítica

- EL SISTEMA DEBERÁ medir proyectos iniciados, completados, cotizados y convertidos.
- EL SISTEMA DEBERÁ medir errores de cálculo reportados, sustituciones y recompra.
- LA ANALÍTICA NO DEBERÁ exponer fotografías ni datos personales innecesarios.
