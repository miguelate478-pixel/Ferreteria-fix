# UX/UI

## Dirección

La aplicación debe sentirse como una mesa de trabajo editorial y técnica, no como una tienda virtual genérica.

## No permitido

- Home llena de tarjetas de producto.
- Carruseles promocionales como elemento dominante.
- Gradientes morado/azul asociados a interfaces genéricas de IA.
- Dashboard de tarjetas KPI repetitivas para el cliente.
- Uso decorativo excesivo de bordes redondeados.

## Componentes distintivos

- Ambiente Canvas.
- Regla Viva de medidas.
- Pila de Color.
- Medidor de Cobertura.
- Ensamblador de Envases.
- Pulso de Stock.
- Ficha de Decisión.
- Libro de Materiales.
- Comparador de propuestas.

## Accesibilidad

- Objetivo WCAG 2.2 AA.
- Navegación completa por teclado.
- Foco visible.
- No comunicar estados solo mediante color.
- Respetar `prefers-reduced-motion`.
- Texto de interfaz en español claro, sin jerga técnica innecesaria.
