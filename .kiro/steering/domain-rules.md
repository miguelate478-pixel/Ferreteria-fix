# Reglas de dominio

## Área

Para ambiente rectangular:

- piso = largo × ancho
- paredes brutas = 2 × (largo + ancho) × altura
- aberturas = suma(ancho × alto × cantidad)
- paredes netas = paredes brutas − aberturas − exclusiones

## Rendimiento

- rendimiento efectivo = rendimiento nominal × factor de superficie × factor de aplicación
- litros base = área pintable × manos ÷ rendimiento efectivo
- litros finales = litros base × (1 + desperdicio/100)

## Validaciones

- Dimensiones, manos y rendimiento mayores que cero.
- Desperdicio entre 0 % y 50 %.
- Las aberturas no pueden superar el área de la superficie.
- El producto debe ser compatible con el uso y la superficie.
- El color debe ser reproducible en la base y presentación seleccionadas.
- El cálculo de cada alternativa usa su propio rendimiento y número mínimo de manos.

## Sustituciones

Una sustitución debe comparar uso, superficie, acabado, base tintométrica, rendimiento, presentación, disponibilidad y precio. Debe señalar diferencias explícitamente.
